# 💙 UniBlue

**UniBlue** é um sistema de gestão financeira desenvolvido para auxiliar repúblicas estudantis no controle de despesas, contas e informações dos moradores.

O projeto foi criado com o objetivo de facilitar a organização financeira compartilhada, promovendo transparência, controle e uma melhor divisão dos gastos entre os estudantes.

---

## 📌 Funcionalidades

### 👨‍🎓 Módulo de Estudantes

* Cadastro de estudantes
* Listagem de estudantes cadastrados
* Atualização de informações
* Exclusão de estudantes

### 💸 Módulo de Gastos

* Registro de despesas da república
* Consulta de gastos cadastrados
* Atualização de registros
* Exclusão de gastos
* Controle do histórico de despesas

### 🧾 Módulo de Contas

* Cadastro de contas fixas
* Controle de vencimentos
* Atualização de informações
* Exclusão de contas
* Acompanhamento de contas pendentes e pagas

### 📊 Módulo de Relatórios

* Relatório de estudantes cadastrados
* Relatório de gastos realizados
* Relatório de contas registradas
* Resumo financeiro da república

### ℹ️ Módulo Sobre o Sistema

O módulo Sobre o Sistema apresenta informações gerais sobre o UniBlue, incluindo sua finalidade, objetivos e funcionalidades disponíveis.

Neste módulo, o usuário pode consultar:

* Objetivo do sistema
* Informações sobre o projeto
* Descrição dos módulos disponíveis
* Versão do sistema
* Créditos dos desenvolvedores
* Informações de uso

---

## 🎯 Objetivo

O UniBlue foi desenvolvido para ajudar estudantes que compartilham uma moradia a manterem um controle eficiente de suas despesas, evitando desorganização financeira e facilitando a divisão de responsabilidades.

---

* Python
* Estruturas de Dados (Dicionários e Listas)
* Programação Estruturada

## 🚀 Como Executar

1. Certifique-se de ter o Python instalado.
2. Clone ou baixe este repositório.
3. Execute o arquivo principal:

```bash
python main.py
```

4. Utilize o menu principal para navegar entre os módulos.

## 📖 Sobre o Nome

**UniBlue** combina os termos:

* **Uni** → Universidade / Universitários
* **Blue** → Referência à expressão financeira "estar no azul", indicando estabilidade e controle financeiro.

O nome representa o objetivo principal do sistema: ajudar estudantes a manterem suas finanças organizadas e sempre no azul.
